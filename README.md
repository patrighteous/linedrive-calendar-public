# linedrive-calendar-public

Public iCal feeds for Pat Acuña's LineDrive calendar system. Rendered by
[`calendar_projector`](https://github.com/) from authoritative event records
in the private LineDrive ecosystem and served via GitHub Pages for
Family Wall to subscribe to.

## Feeds

- `oldline.ics` — business operations
- `family.ics` — household-facing events
- `personal.ics` — Pat-only events

All feeds are read-only. Do not edit these files directly; edit the source
records in the LineDrive ecosystem, then re-run the projector.

**Governance:** `ecosystem_stack/reference/operations/calendar/CALENDAR_DIRECTIVE.md` (private repo)
